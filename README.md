# Onepager B2E

Ein schlanker Onepager für **Business to Environment (B2E)** basierend auf der Broschüre. 

## Inhalte
- Hero mit Claim und Call-to-Action
- Sektionen für Vision, Werte, solidarisches Finanzierungsmodell, Vorteile
- Kontaktbereich mit Mail-Link und Formular (mailto: kontakt@hip.swiss)

## Anpassungen
- Farben und Typografie werden in `styles.css` definiert.
- Inhalte können in `index.html` bearbeitet werden.
- Bilder/Logos können bei Bedarf unter `assets/` ergänzt und im HTML referenziert werden.
